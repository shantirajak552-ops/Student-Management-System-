# Student Management System

This is a beginner-friendly Python console-based project.

## Features
• Add student name and marks
• Show student list
• Check pass/fail result
• Menu-driven program using while loop

## Pass / Fail Rule
• Marks >= 33 : PASS
• Marks < 33 : FAIL

## Technologies Used
• Python

## How to Run
1. Install Python (version 3)
2. Save the file as student.py
3. Run the program using:
   python student.py

## Learning Outcome
This project helped me understand lists, loops, user input, and conditional statements in Python.
